Specify the name for the Fortify Project Results file (for example, audit.fpr). If no file is specified and you are running a Fortify SCA scan, the analysis results are written to scan.fpr in the workspace. (If this file already exists, it is overwritten.) If no file is specified and you are not running a Fortify SCA scan and you are uploading results to Fortify Software Security Center, the plugin searches (./**/*.fpr) in the workspace for the FPR file with the latest modified date.
