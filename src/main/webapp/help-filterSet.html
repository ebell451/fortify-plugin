A filter set determines which issues are displayed. You can select a filter set for an application that exists on the server. Leave the field blank to use the default filter set configured in Fortify Software Security Center.

Fortify Security Center provides filter sets based on the issue template configured for the application. By default, every issue template includes the Quick View and Security Auditor View filter sets. Your organization might have additional filter sets.

The build failure criteria, the NVS calculation, and the Fortify Assessment page depend on the issues filtered by the filter set. For example, if a Quick View filter set is applied and no issues are found, then the build failure criteria determines that there is no reason to set this build to unstable and the NVS is set to zero. The graph summary also shows zero.
